# 在线工具箱 - 功能需求表格

## 概述
本网站是一个集合了多种实用在线工具的静态网站，采用现代化的深色主题设计，提供简洁美观的用户体验。

## 设计特点

### UI设计
- 深色主题（Dark Mode），护眼舒适
- 渐变色彩搭配，现代感强
- 卡片式布局，层次分明
- 响应式设计，适配移动端
- 悬停动画效果，交互友好

### 技术实现
- 纯HTML/CSS/JavaScript，无依赖框架
- CSS变量管理主题样式
- Grid布局实现响应式网格
- 模块化JavaScript代码
- 本地运行，无网络依赖

## 文件结构

```
Tools/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # 脚本文件
└── 功能需求表格.md      # 本文档
```

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可使用
2. 点击顶部导航栏切换不同工具分类
3. 在各个工具卡片中输入内容进行操作
4. 所有功能均在本地运行，无需联网
