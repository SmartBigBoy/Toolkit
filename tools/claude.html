国内安装 Claude Code 完整指南

解决国内网络环境下的 Claude Code 安装痛点，从 Node.js 到 cc-Switch 配置，一篇到位。

适用于 Windows · 基于 Deepseek 模型 · 2026-05
1 安装 Node.js

Claude Code 基于 Node.js 运行，首先安装 Node.js v24.15.0。

下载地址：
https://nodejs.org/dist/v24.15.0/node-v24.15.0-x64.msi

下载完成后双击安装包，一路点击 Next 完成安装即可。

安装完成后，打开命令行（Win + R，输入 cmd，回车），验证安装是否成功：

node -v

成功将返回版本号。接着验证 npm：

npm -v
✔ 正常输出 — 如果两个命令都返回了版本号，说明 Node.js 和 npm 已安装成功。
2 安装 Claude Code

使用 npm 全局安装 Claude Code，指定国内镜像源加速下载：

npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com

等待安装完成后，验证版本：

claude --version
✔ 正常输出 — 返回 Claude 版本号即安装成功。
3 安装 cc-Switch

cc-Switch 是一个模型切换工具，用于将 Claude Code 的底层模型切换为 Deepseek 等国内可用的模型。

下载地址：
https://github.com/farion1231/cc-switch/releases/download/v3.15.0/CC-Switch-v3.15.0-Windows.msi

下载完成后双击安装，按照向导提示完成即可。

4 获取 Deepseek API Key

进入 Deepseek 官网 注册账号，登录后充值金额（初次建议充值 10 元即可）。

在控制台中点击 API Key，创建一个新的 API Key。复制生成的一串字符——这个 Key 就是外部软件访问 Deepseek 模型的凭证，里面包含了你的身份信息，调用模型时官方会计费到你的账户。

⚠ 注意事项
• API Key 仅在创建时出现一次，复制后请妥善保存
• 如果忘记了，可以删除后重新创建，旧的 Key 会立即失效
• 建议给 API Key 添加描述名称，方便知道哪些工具在调用 Deepseek
5 配置 cc-Switch

cc-Switch 安装完成后，打开软件：

  1. 在右上角点击 ＋ 按钮，进入模型配置页面
  2. 选择 DeepSeek 作为模型提供商
  3. 下滑找到 API Key 输入框，粘贴刚才复制的 Key
  4. 点击 添加 保存配置
6 体验 Claude Code

所有配置完成后，在命令行中输入：

claude

将会进入 Claude Code 交互界面。一直按 Enter，等到出现输入光标后，就可以开始使用了。比如你可以先问它：

你使用的什么模型？你能帮我做什么？
7 Claude Code 常用快捷键

熟练使用快捷键能大幅提升操作效率，以下是常用快捷键一览：

最常用

Enter发送消息
Ctrl + J换行（而非发送）
Ctrl + T切换任务列表
Ctrl + O切换 Transcript
Ctrl + S保存 / 暂存对话
Esc取消 / 关闭弹窗
Ctrl + G打开外部编辑器
Ctrl + L清空输入框

对话与历史

↑ / ↓上 / 下一条历史输入
Ctrl + R搜索命令历史
Ctrl + Z / Ctrl+Shift+Z撤销 / 重做输入

模型控制

Win+P / Cmd+P切换模型
Win+O / Cmd+O切换快速模式
Win+T / Cmd+T切换思考模式
Shift + Tab循环切换输入模式

确认对话框

Y / Enter确认
N / Esc拒绝
Space切换选项
Tab / ↑↓切换焦点

任务与 Agent

Ctrl + B当前任务放到后台
Ctrl+X → Ctrl+K杀掉所有 Agent

界面导航

Ctrl+U / Ctrl+D上半页 / 下半页滚动
Ctrl+B / Ctrl+F上一页 / 下一页
G / Shift+G跳到顶部 / 底部
Ctrl+Shift+C复制选中内容
